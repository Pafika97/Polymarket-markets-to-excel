# Polymarket ➜ Excel (3 колонки)

Скрипт загружает **активные рынки** с Polymarket и создаёт Excel с 3 колонками:

- **Название рынка**
- **Параметр A**
- **Параметр B**

По умолчанию берутся **только бинарные** рынки (Yes/No). Можно включить многовариантные рынки флагом `--include-multi`
— в таком случае у них будут взяты **первые два** исхода.

## Быстрый старт

1) Установите зависимости (желательно в отдельном venv):
```bash
pip install -r requirements.txt
```

2) Запустите скрипт и получите Excel:
```bash
python main.py --output markets.xlsx
```

Опционально, включить многовариантные рынки:
```bash
python main.py --output markets.xlsx --include-multi
```

## Что внутри Excel

- Лист `Polymarket` — таблица из трёх колонок.
- Лист `Meta` — источник и время генерации (UTC).

## Примечания

- Скрипт использует публичный API `gamma-api.polymarket.com`. Если у Polymarket меняется форма ответа,
  скрипт старается извлечь название/исходы по нескольким возможным ключам.
- Для scalar/range рынков в большинстве случаев Polymarket всё равно хранит бинарные исходы *Yes/No*,
  а порог/условие отражён в названии рынка (например, «Цена BTC к 31 декабря > 105000?»).
